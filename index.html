Fractic

This full-featured Progressive web and was built in Ionic and exported to Google play app store. It can be run in a wide range of browsers and on Android devices from SDK version 19.

Some of the most prominent challenges were.

  • Making sure cross-browser compatibility is handled well particularly with IOS. Graceful fails for unimplemented native features in both Chrome Safari and Firefox. Significant workarounds for Safari were required.
  • Consistently handling all types of input from touch gestures, mouse, scroll wheel, double clicks.
  • WebGL with graceful fallback to javascript canvas. Passing data to shaders and ensuring the WEBGL algorithm matched the CPU javascript algorithms.
  • Studding CSS and SASS to ensure a wide range of browsers were supported.
  • Server-side PHP code to enable sharing and saving of Fractals, using rich link metadata.
  • Custom made UI built in HTML and SASS, providing unique solutions such as the colour stop gradient and histogram tool.

APOS

This was the first feature-complete application I created. An Electronic Point Of Sale system for my business Apache. Built-in Java with a Swin UI and ORM using Hibernate to an embedded Derby database. Featuring a front end for a touch screen interaction, Hardware support for cash drawer and receipt printing. The back end built for mouse and keyboard.

Features include

  • Staff hour tracking
  • Products by catagory, department, size colour and other searchable metrics.
  • VAT and financial reports
  • Security levels
  • Audit Trails
  • Product and sales reporting
  • End of day reporting
  • Cash reconciliation.
  • All other basic features you would expect from an EPOS system.
