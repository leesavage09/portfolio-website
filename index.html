Fractic

Fractic allows the users to explore and colour the amazing world of fractals, such as the Mandelbrot set.

Some of the most prominent challenges were:

  • Ensuring cross-browser compatibility is handled well. Graceful fails for unimplemented native features in Chrome, Safari and Firefox
  • Handling all types of input from touch gestures, mouse, scroll wheel and double clicks
  • Implementing GPU shaders with graceful fallback to CPU rendering
  • Passing scientific data to shaders and ensuring the output of the GPU algorithm matched the CPU algorithms precisely
  • Server-side code to enable sharing and saving of Fractals using rich link metadata
  • A sophisticated custom-built UI including the colour stop gradient and histogram tool
